# 🌿 Indian Plant Identification System with AI

An AI-based web application that identifies Indian medicinal plants from images and provides useful information about them.

## 🚀 Features
- Image-based plant identification
- AI-powered prediction and information system
- Displays plant details and uses
- Simple and user-friendly interface

## 🛠 Tech Stack
- Python
- Flask
- TensorFlow / Keras
- Streamlit

## ▶️ Run the Project
```bash
git clone https://github.com/your-username/Indian-plant-identification-system-with-AI-information-system.git
cd Indian-plant-identification-system-with-AI-information-system
pip install -r requirements.txt
cd /backend npm run dev
streamlit run app.py
